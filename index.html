About NAKL

NAKL is an open source Vietnamese input keyboard for Mac OS. Its engine is based on XVNKB, a well known input keyboard on Linux.

Current Version

Latest stable version is NAKL 0.1. You can also build NAKL from source with Xcode 4.0 and later.

Features

  • Lightweight
  • Free and open
  • Faster typing with keyboard shortcuts
