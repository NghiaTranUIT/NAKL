About NAKL

NAKL is an open source Vietnamese input keyboard for Mac OS. Its engine is based on XVNKB, a well known input keyboard on Linux.

Current Version

The first version of NAKL (0.1) is still under development. If you want to try out NAKL, feel free to contact its author or build NAKL from source.

Features

  • Lightweight
  • Free and open
  • Faster typing with keyboard shortcuts
